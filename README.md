## Sobre o projeto

O **StructBoost** foi desenvolvido com o objetivo de reforçar como uma **estrutura HTML sólida e semântica** facilita o processo de criação de interfaces modernas.  
Durante o aprendizado no **Codeboost**, o foco deste projeto foi entender como um bom HTML permite que o CSS flua naturalmente, deixando o layout mais organizado, flexível e escalável.


## Tecnologias utilizadas

- **HTML5** – Estrutura semântica limpa  
- **CSS3** – Estilização visual  
- **CSS Keyframes** – Animações personalizadas  
- **JavaScript no HTML** – Interatividade usando uma biblioteca externa

## Objetivos do projeto
- Entender a importância da arquitetura HTML
- Criar uma landing page moderna do zero
- Aplicar boas práticas de estruturação
- Utilizar animações através de @keyframes
- Desenvolver interatividade com JavaScript diretamente no HTML
- Colocar em prática os aprendizados do curso Codeboost
